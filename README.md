# Trustworthy Privacy Preserving Framework in Industrial IoT Systems

A security framework designed to ensure data privacy and trust in Industrial Internet of Things environments using advanced cryptographic techniques.

## Tech Stack
Python, Blockchain, SHA-256/SHA-512, IoT Protocols, Edge Computing, Smart Contracts

## Key Features
- Privacy-preserving data aggregation for IIoT sensors using homomorphic encryption
- Blockchain-based trust management to prevent data tampering
- Lightweight authentication protocol for resource-constrained IoT devices
- Reduced security overhead by 25% compared to traditional frameworks
- Secure communication between industrial sensors and cloud servers
